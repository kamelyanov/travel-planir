# Travel Planner

Приложение для планирования путешествий с возможностью сохранения маршрутов и отслеживания времени.

Проект выполнен с использованием ИИ-ассистента Qwen3.5 (Alibaba Cloud)

https://kamelyanov.github.io/travel-planir/

## Функционал

- Планирование маршрутов с указанием дат и времени
- Сохранение данных в localStorage
- Отображение списка запланированных маршрутов
- Валидация форм
- Визуальный редактор с возможностью перетаскивания блоков
- Таймлайн с автоматическим выстраиванием по времени
- Карточки с информацией о локациях и переездах
- Группировка элементов
- Темная тема с возможностью рисования от руки

## Структура проекта

```
src/
├── models/                 # Бизнес-модели
│   ├── Trip.js             # Модель поездки
│   ├── LocationBlock.js    # Модель локации
│   ├── Transfer.js         # Модель переезда
│   ├── EventGroup.js       # Модель группировки
│   └── Route.js            # Старая модель маршрута (для совместимости)
├── views/                  # Компоненты UI
│   ├── RouteForm.js        # Форма маршрута
│   ├── RouteList.js        # Список маршрутов
│   └── MapView.js          # Вид карты
├── controllers/            # Контроллеры
│   ├── RouteController.js  # Контроллер маршрутов
│   └── TripController.js   # Контроллер поездок
├── services/               # Сервисы
│   ├── StorageService.js   # Сервис хранения
│   ├── MapService.js       # Сервис карт
│   └── WeatherService.js   # Сервис погоды
├── utils/                  # Утилиты
│   ├── validators.js       # Валидаторы
│   ├── formatters.js       # Форматтеры
│   └── helpers.js          # Вспомогательные функции
└── main.js                 # Точка входа
```
