# Travel Planner

Приложение для планирования путешествий с возможностью сохранения маршрутов, визуального редактирования и отслеживания времени.

Проект выполнен с использованием ИИ-ассистента Qwen3.5 (Alibaba Cloud)

🌐 **Демо:** https://kamelyanov.github.io/travel-planir/

## 🚀 Технологический стек

- **Frontend:** React 18
- **Стейт-менеджмент:** Zustand
- **Стилизация:** Tailwind CSS
- **Сборка:** Vite
- **Анимации:** Framer Motion
- **Иконки:** Lucide React
- **Работа с датами:** date-fns
- **Генерация ID:** nanoid

## ✨ Функционал

- 🗺️ Планирование маршрутов с указанием дат и времени
- 💾 Сохранение данных в localStorage
- 📋 Отображение списка запланированных маршрутов
- ✅ Валидация форм
- 🎨 Визуальный редактор с возможностью перетаскивания блоков
- ⏱️ Таймлайн с автоматическим выстраиванием по времени
- 📍 Карточки с информацией о локациях и переездах
- 📦 Группировка элементов
- 🌙 Темная тема с возможностью рисования от руки
- 🔗 Связи между точками маршрута (стрелки с подписями)
- 🎯 Различные типы транспорта (пеший, самолет, поезд, автобус, авто)

## 📦 Модели данных

- **Trip** — поездка (id, название, даты начала/окончания)
- **LocationBlock** — локация (время прибытия/отправления, длительность, заметки)
- **Transfer** — переезд (откуда-куда, время, тип транспорта, стоимость)
- **EventGroup** — группировка (объединение нескольких блоков в рамку)

## 🏗️ Структура проекта

```
├── models/                 # Бизнес-модели
│   ├── Trip.js             # Модель поездки
│   ├── LocationBlock.js    # Модель локации
│   ├── Transfer.js         # Модель переезда
│   └── EventGroup.js       # Модель группировки
├── views/                  # React-компоненты UI
│   ├── RouteForm.js        # Форма маршрута
│   ├── RouteList.js        # Список маршрутов
│   └── MapView.js          # Вид карты
├── controllers/            # Контроллеры
│   ├── RouteController.js  # Контроллер маршрутов
│   └── TripController.js   # Контроллер поездок
├── services/               # Сервисы
│   ├── StorageService.js   # Сервис хранения
│   ├── MapService.js       # Сервис карт
│   └── WeatherService.js   # Сервис погоды
├── utils/                  # Утилиты
│   ├── validators.js       # Валидаторы
│   ├── formatters.js       # Форматтеры
│   └── helpers.js          # Вспомогательные функции
└── main.js                 # Точка входа
```

## 🛠️ Запуск проекта

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Предпросмотр продакшен-сборки
npm run preview
```
